# 📊 Customer Shopping Behavior Analysis

An end-to-end data analytics project covering Python-based exploratory analysis, SQL business querying, Power BI dashboarding, and a final presentation report.

## 🧾 Overview

This project analyzes 3,900+ customer transactions to uncover revenue drivers, product performance, customer segments, and shopping behavior patterns. The workflow follows a real-world analytics pipeline:

**Python (EDA) → SQL (Business Analysis) → Power BI (Dashboard) → Report & Presentation**

📄 [View Full Report](./Report_Analysis.pdf) · 📊 [View Presentation](./PPT.pdf) · 📈 [View Dashboard](./Dashboard.pdf)

## 📂 Dataset

- 3,900 rows, 18+ features
- Customer demographics (age, gender)
- Product details and categories
- Subscription status
- Review ratings
- Discounts and shipping type
- Purchase amount

## 🛠 Tools & Technologies

- **Python:** pandas, numpy, matplotlib
- **SQL:** PostgreSQL / MySQL / SQL Server-compatible queries — aggregations, joins, segmentation
- **Power BI:** KPI cards, charts, slicers, interactive dashboard
- **Reporting:** PDF report and presentation deck

## 🔧 Project Workflow

**1. Data Preparation (Python)**
- Loaded dataset using pandas; performed summary statistics and EDA
- Cleaned missing values and created new features (e.g., age groups)
- Exported cleaned data for SQL analysis

**2. SQL Analysis**
Answered specific business questions, including:
- Revenue by gender and by age group
- High-spend vs. discount-driven customer behavior
- Highest-rated products
- Standard vs. express shipping preferences
- Subscriber vs. non-subscriber spending behavior
- Customer segmentation (Loyal, Returning, New)
- Repeat buyer patterns

**3. Power BI Dashboard**
Built an interactive dashboard featuring:
- Total customers and average purchase amount
- Review rating trends
- Revenue by category and by age group
- Subscription status breakdown (donut chart)
- Filters for gender, category, and shipping type

📊 [View Dashboard Export](./Dashboard.pdf)

**4. Final Report & Presentation**
- 📄 [Report_Analysis.pdf](./Report_Analysis.pdf) — full written analysis with all business questions and insights
- 📊 [PPT.pdf](./PPT.pdf) — recruiter-friendly presentation summarizing key findings

## 📈 Key Insights

- Male customers spent more per person on average ($71.48 vs. $44.47) and, combined with a larger customer base (57%), generated 2.1x total revenue
- Young Adults generated the highest revenue among all age groups
- Loyal customers made up the vast majority of the customer base (3,116 of 3,900) — indicating strong retention, though the New customer segment remains small (83)
- Gloves, sandals, and boots were the highest-rated product categories
- Discount usage was concentrated in a small number of product categories rather than spread evenly
- Subscribers and non-subscribers spent similar average amounts — subscription status alone did not predict spend
